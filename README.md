Build instructions
------------------

  - (Optional) Edit values in `app/secure.gradle`.
  - Build app (Linux, MacOS): `./gradlew assembleDebug`.
  - Build app (Windows): `gradle.bat assembleDebug`.
  - Install apk: `adb install app/build/outputs/apk/app-debug-unaligned.apk`.
